# Travelers-Kingdom
Travel-Kingdom: Responsive travel agency landing page with HTML5, CSS3 &amp; vanilla JS. Fixed nav (Home/About/Contact) + mobile hamburger, expandable destination search, hero with full-screen bg image, social icons, adventure headline, cultural description &amp; "Book Now" CTA. Flexbox responsive design.


BRIEF EXPLANATION

Travelers Kingdom is a sleek, responsive landing page for a travel agency, built with HTML5, CSS3, and vanilla JavaScript. It features a fixed navigation bar linking Home, About Us, and Contact Us pages, plus an interactive search bar for destinations and keywords.

Main Sections
The hero section showcases a stunning full-screen background image with an adventure headline, cultural exploration description, social media icons, and eye-catching "Book Now" button. Fully mobile-optimized using flexbox, smooth CSS transitions, and hamburger menu for smaller screens.

Key Features
Fixed blue nav bar with responsive mobile toggle

Expandable search functionality ready for API integration

Engaging hero content with overlay for text readability

Clean Poppins font and Unicons for professional polish
